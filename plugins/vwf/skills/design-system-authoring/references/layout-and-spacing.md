# Layout & Spacing

- **Spacing scale** — a single base unit and the steps built on it (e.g.
  4-point: 4 / 8 / 12 / 16 / 24 / 32 / 48 / 64). Everything spaces on the scale.
- **Grid & containers** — column count, gutter, and max content width.
- **Breakpoints** — the canonical widths and the **behavioral intent** at each
  (what stacks, collapses, or reveals). The widths are a contract; the media
  query is realization.
- **Radius scale** — corner-radius steps (none / sm / md / lg / full) and where
  each applies.
- **Elevation** — the shadow/elevation steps and their meaning (raised, overlay,
  modal) — not the exact box-shadow CSS.

Define only the steps in use.
