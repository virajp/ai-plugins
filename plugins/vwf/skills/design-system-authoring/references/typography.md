# Typography

Pin the **font pairing** and a **type scale** — both product decisions.

- **Families** — heading, body, and mono (if used), named (e.g. specific Google
  Fonts) with the mood/intent each conveys. Naming the family is a contract;
  loading it is realization.
- **Type scale** — each step: size / line-height / weight / letter-spacing, and
  what it is for (display, h1–h3, body, small, caption). Use a consistent ratio.
- **Responsive behavior** — how the scale shifts at smaller breakpoints (the
  intent, not the media query).
- **Rules** — max line length for body text, minimum body size, and tabular/
  numeric usage if relevant.

Keep the scale to the steps the product actually uses.
