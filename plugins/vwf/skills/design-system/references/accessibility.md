# Accessibility Standard

State the bar the product **commits to** — this is gated, not aspirational.

- **Target** — the conformance level (e.g. WCAG 2.2 AA).
- **Contrast** — text ≥ 4.5:1; large text and UI/graphics ≥ 3:1.
- **Keyboard** — every interactive element reachable and operable; visible
  focus; logical order; no traps (except an intentional modal focus-trap).
- **Focus management** — focus moves into an opened dialog and back on close;
  route changes move focus to the heading.
- **Targets** — minimum hit area (e.g. 24×24, or 44×44 for touch).
- **Semantics** — headings in order, labelled controls, status messages
  announced, images have text alternatives.
- **Motion & media** — honor reduced-motion; no autoplay audio/video without a
  control.

Every screen inherits this standard; deviations require an explicit, recorded
exception.
